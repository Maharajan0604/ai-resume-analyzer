# AI Resume Analyzer

An AI-powered resume analyzer built with **React**, **Vite**, and **Puter.js**.
Upload your resume and receive instant AI feedback.

## Features
- Resume upload (PDF)
- AI-powered feedback
- Serverless architecture using Puter
- Modern React + Vite setup

## Tech Stack
- React
- Vite
- Zustand
- Tailwind CSS
- Puter.js
